What is "normal flow" anyway?

I know you came here to learn about how to position elements on a page using CSS, but before we get into that it's important to understand what we mean when we talk about normal flow. Simply put, Normal Flow is just the default way that elements are laid out on a page without any additional CSS styling. In practice that means that (in English at least...) block elements (e.g. paragraphs, divs) will start from the top and will be seperated onto new lines and that inline elements (e.g. spans) will display from left to right without being moved to a new line.

Fixed It

Fixed positions are also removed from the normal document flow but are set relative to the viewport. So the element will always stay in the same place on-screen no matter where the page is scrolled. Fixed elements can be used for things like scroll bars or navigation where you want them to always remain in view.

Bonus Round: Let's get sticky

Sticky positions will initially behave just like relative positions, but they do something special once a certain scroll point is reached - they toggle to a fixed position instead and will remain in that position to accompany you on your journey scrolling through the page.
