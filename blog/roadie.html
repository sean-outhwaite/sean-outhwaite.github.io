Hit the Road

Roadie is an AI powered roadtrip planning app. Once you've signed up, you can give it some information about where you're going and what you're interested in and Roadie will generate a route for you to take with some recommended stops along the way.

You can save your trips, and access them at any time if you're signed in.

Stacks

Roadie is a React app built in TypeScript, using Auth0 to handle sign up/authentication and a SQLite database to store user and roadtrip information.

It uses the Gemini API to find potential stops between your trip start and destination locations based on the interests you select. We then take that information and generate a route using the Google Maps APIs, display the path on a map, and populate some data from Google Places.

The frontend styling was built with Tailwind & Shadcn.
