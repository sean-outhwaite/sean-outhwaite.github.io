It's alive!

I got a bit obsessed with making updates to the blog generator and I think it's now at state where I'm functionally done with it for now(could still use some styling though).

I've reformatted the output HTML into a textarea so it's a bit more contained and have added a copy button that adds the HTML to your clipboard.

In more exciting news, you can now edit all elements of the blog directly in the preview.

So if you make a typo, or want to add/remove some text you can just click into the preview and make your changes. Once the preview loses focus it will save those changes and update the output HTML.

Why Bloggen?

I've also given it a new name: Bloggen. When I first started working on it, I created the initial HTML & JS files and was too lazy to type out the full "blogGenerator" or even use camelcase to separate "blog" from "gen".

Since then, every time I see those filenames I have been reading them as "Bloggin" which I think sounds fun and also now I can justify not using camelcase in those file names (don't worry, all of the variables do actually use camelcase).
